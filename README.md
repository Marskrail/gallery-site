# 🎨 Онлайн Галерея

Красивая галерея изображений с поиском по тегам, работающая полностью в браузере.

## ✨ Особенности

- 🌐 **Работает онлайн** - не требует сервера
- 🔍 **Поиск по тегам** - мгновенная фильтрация
- 🏷️ **Система тегов** - как в популярных галереях
- 💾 **Локальное сохранение** - данные хранятся в вашем браузере
- 📱 **Адаптивный дизайн** - работает на всех устройствах
- 🎨 **Красивый интерфейс** - современный дизайн в синих тонах

## 🚀 Быстрый старт

1. Откройте `index.html` в браузере
2. Нажмите "Показать все" чтобы увидеть изображения
3. Используйте поиск или кнопки тегов
4. Добавляйте свои изображения через форму

## 📁 Структура проекта

- `index.html` - главная страница галереи
- Все данные хранятся в LocalStorage браузера

## 🌐 Деплой на GitHub Pages

1. Залейте файлы в репозиторий
2. Включите GitHub Pages в настройках репозитория
3. Сайт будет доступен по адресу: `https://ваш-логин.github.io/название-репозитория`

## 🛠️ Технологии

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (ES6+)
- LocalStorage для хранения данных
